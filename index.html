Defining the cell and grid sizes with this text below

This is an attempt to build a modern typographic grid, from bottom up, starting with setting the font — up to defining grid cells and gutters both vertically and horizontally.

The recipe is simple: a grid cell width is around 40-50 characters, a grid cell height is 4 lines of text. The gutter height equals the height of a single line.
